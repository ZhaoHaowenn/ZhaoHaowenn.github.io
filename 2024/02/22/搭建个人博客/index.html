一，安装Node.js/git/hexo

Node.js下载地址：Node.js 

npm install -g hexo-cli

二，初始化博客

cd xxx\_blog
hexo init

三，常用命令

  • 新建一篇文章，默认使用_config.yaml中default_layout参数指定的布局
hexo new \[layout\] <title>
  • 生成静态文件
hexo g
  • 启动本地调试服务器
hexo s
  • 部署到远程网站，部署前请确保已经安装Git插件并已经按照上述完成了_config.yaml中的相关配置
hexo d
  • 清除缓存文件db.json和已生成的静态文件夹public
hexo clean
